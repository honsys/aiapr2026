> Use /chat save <path> to save your current conversation to a file:
/chat save my-session.json

This exports the full conversation as JSON — message history, context files, agent config, etc. To reload it later:

/chat load my-session.json

A few useful options:
- /chat save backup.json --force — overwrite an existing file
- /chat save ~/backups/session.json — save to a specific path

Note: conversations are also auto-saved to a database after every turn (per-directory), so you can resume 
without manually saving using kiro-cli chat --resume or /chat resume.
