
inspect the c++26 impl of the gem lang scientific and quantitative finace runtime and all builtin functions and objects and extend it with features to support astrophysics and solar physics and planetary sciences


## What was built

### New file: gem_src/gem_astro.hpp

A complete GemAstro class (registered as astro builtin) with:

Constants (as properties): G, c, AU, pc, ly, Msun, Rsun, Lsun, Tsun, Mearth, Rearth, H0, sigma_sb

Unit conversions: to_ly(), to_pc(), to_au(), deg_to_rad(), rad_to_deg()

Stellar physics:
- luminosity(R_rsun, T) — L/Lsun via Stefan-Boltzmann
- stefan_boltzmann(T, R) — absolute luminosity in W
- wien(T) — peak wavelength
- abs_magnitude(m, d_pc) — distance modulus conversion
- spectral_class(T) — O/B/A/F/G/K/M classification
- schwarzschild_radius(M) — black hole event horizon
- escape_velocity(M, R)

Orbital mechanics:
- orbital_period(a_au, M_msun) — Kepler's 3rd law
- orbital_velocity(a_m, M_kg)
- hill_sphere(a, mp, ms), roche_limit(R, rho_p, rho_s), synodic_period(p1, p2)
- planet(name) — returns object with mass/radius/semi_major/period/moons for all 8 planets

Solar physics:
- solar_flux(dist_au) — irradiance at any distance
- solar_wind_pressure(n_cm3, v_km_s) — dynamic pressure in nPa
- sunspot_cycle(year) — approximate SSN based on Schwabe cycle
- parker_spiral_angle(v_sw, dist_au) — IMF spiral angle
- solar_activity() — current cycle 25 status object

Cosmology:
- hubble_distance(z), redshift_velocity(z)
- lookback_time(z) — numerical ΛCDM integration in Gyr
- critical_density()

Coordinate transforms:
- equatorial_to_galactic(ra, dec) — returns {l, b} object
- angular_separation(ra1, dec1, ra2, dec2) — great-circle distance

Exoplanet / planetary science:
- transit_depth(rp_rearth, rs_rsun) — photometric transit depth
- habitable_zone(L_lsun) — returns {inner_au, outer_au}
- equilibrium_temp(L, d_au, albedo) — planetary equilibrium temperature

Pulsar / neutron star:
- pulsar_spindown(P, Pdot) — returns {age_yr, Bfield_G, edot_W}
